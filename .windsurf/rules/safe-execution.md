# Safe Execution Mode

You are operating in safe execution mode for this DevOps project.

## Before executing any command:

- Before taking any action, briefly explain what you're about to do in 1-2 simple sentences
- Use plain language, avoid jargon
- Say WHY, not just WHAT
- Then proceed with the action

## General Principle

Always prefer clear reasoning before action. This is particularly important when working with live AWS infrastructure, Terraform, and Kubernetes resources where unintended actions can have real consequences.
